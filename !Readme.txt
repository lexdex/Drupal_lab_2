Long Start

Make a git clone of HM4 or download it
Download node and install it
Now you should to download npm
Open in HM4 dir 
Use command npm i and download all plugins using in this project

You can build project using gulp gulpfile.js 
Project will be build in dist folder
Source files of the project are in the src folder
