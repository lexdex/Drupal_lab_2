Fast Start
Make a git clone of HM4 or download it
Open dir HM4/homework/src/index.html

-------------------------------------------
Long Start

Make a git clone of HM4 or download it
Download node and install it
Now you should  download npm

Open in HW4 directory
Use command npm i and download all plugins used in this project

You can build project using gulp gulpfile.js 
This project will be build in dist folder
Source files of the project are in the src folder
